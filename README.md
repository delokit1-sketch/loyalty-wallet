# Loyalty Wallet

Сервис для малого бизнеса, который позволяет выпускать виртуальные дисконтные и бонусные карты.
Клиенты могут добавить карту в **Apple Wallet** или **Google Wallet**, получать бонусы и скидки,
а бизнес — отправлять уведомления и акции.

## Этапы проекта
1. Регистрация бизнеса и выпуск карты
2. Начисление бонусов и учёт покупок
3. Push-уведомления и геоакции
4. Тарифы и интеграции
